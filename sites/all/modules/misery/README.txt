INTRODUCTION
------------
Misery is a module designed to make life difficult for certain users.

Project page: http://drupal.org/project/misery.


DEPENDENCIES
------------
This module requires the Manifest module: 
http://drupal.org/project/manifest


INSTALLATION
------------
Install and enable the Misery module.
For detailed instructions on installing contributed modules see:
http://drupal.org/documentation/install/modules-themes/modules-7


USAGE
-----
Use the manifest module to create a list of users that will endure misery.
This module will automatically create a manifest, and you can add users to it,
or delete it and make your own.
